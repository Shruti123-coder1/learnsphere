# 📚 LearnSphere

LearnSphere is a responsive online learning platform built with **React**, **Vite**, and **Tailwind CSS**. It allows users to explore courses, view detailed course information, and manage their learning progress through a modern and responsive interface.

---

## 🚀 Tech Stack

- ⚛️ React
- ⚡ Vite
- 🎨 Tailwind CSS
- 🧭 React Router DOM
- 🟨 JavaScript

---

## ✨ Advanced Tailwind CSS Assignment

### 📖 Feature 1 – Course Details Modal

- Responsive course details modal
- Full-screen overlay
- Accessible close button
- Internal scrolling
- Hover, Focus & Active states
- Responsive on all screen sizes

### 🔔 Feature 2 – Notification Center

- Sticky navbar
- Notification badge
- Responsive notification panel
- Read/Unread notification styling
- Scrollable notification list
- Accessible notification button
