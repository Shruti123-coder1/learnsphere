# 📚 LearnSphere – Responsive Learning Platform

A modern, responsive online learning platform built with **React.js**, **Vite**, and **Tailwind CSS**. LearnSphere allows students to explore courses, track learning progress, search courses, and manage their learning journey through a clean and user-friendly interface.

---

## ✨ Features

- Responsive Navigation
- Hero Section with Call-to-Action
- Learning Statistics Dashboard
- Real-Time Course Search
- Category Filters
- Course Cards with Progress Bars
- Continue Learning Section
- Login & Registration
- User Profile
- Fully Responsive Design
- Custom Theme & Google Fonts

---

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Router DOM
- JavaScript (ES6+)
- Google Fonts (Poppins & Inter)

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/Shruti123-coder1/learnsphere.git
```

### Navigate to the Project

```bash
cd learnsphere
```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```
